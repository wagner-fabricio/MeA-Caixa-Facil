\# Contributing



Thanks for your interest.



This repository provides \*\*operational primitives\*\*. Stability and

predictability take priority over feature growth.



\## General Guidelines



\- Small, focused pull requests only

\- No breaking changes without prior discussion

\- New skills must include:

&nbsp; - A `SKILL.md` file

&nbsp; - Deterministic behavior

&nbsp; - Clear inputs, outputs, and side effects



\## Scope



The following are considered out of scope for this repository:

\- Autonomous decision-making logic

\- Policy enforcement or permissions

\- Agent orchestration frameworks



\## Review Policy



\- Maintainers may decline changes that increase complexity or risk

\- Not all contributions will be accepted
